# gridea-theme-pan
示例：[Miracol.cn](https://miracol.cn/)
请不要修改底部版权连接！谢谢！

## 已有功能
1、图片灯箱效果
2、文章页面目录功能
3、点击加载更多式分页
4、谷歌字体的logo
## 待添加功能（功能已经实现待Gridea添加自定义页面添加！）
1、搜索功能
2、从豆瓣调取已看过的电影和图书功能！
3、足迹功能！
