# gridea-theme-pan
示例：[Miracol.cn](https://miracol.cn/)
请不要修改底部版权连接！谢谢！

1. 图片灯箱效果
2. 文章页面目录功能
3. 点击加载更多式分页
4. 谷歌字体的logo
